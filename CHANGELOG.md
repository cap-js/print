# Change Log

All notable changes to this project will be documented in this file.
This project adheres to [Semantic Versioning](http://semver.org/).
The format is based on [Keep a Changelog](http://keepachangelog.com/).

## Version 0.2.0 - 2025-12-19

### Added

- Multitenancy support
- Support for `@Core.ContentDisposition.Filename` to specify filename

### Fixed

- Draft enabled entities can be print enabled
- Entities with composite keys can be print enabled
- File upload during printing is now formatted correctly

## Version 0.1.0 - 2025-11-28

### Added

- Initial release that provides out-of-the-box integration with the SAP Print Service.
